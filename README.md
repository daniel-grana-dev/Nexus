# Nexus

Self-hosted AI inference platform with hybrid LLM routing, RAG pipeline, and production monitoring.

> Current status: Planification in progress

## Stack
- **Local LLM**: Ollama (Llama 3.2)
- **Cloud LLM**: OpenAI GPT-4o-mini
- **API**: FastAPI + Python
- **Database**: PostgreSQL + pgvector
- **Monitoring**: Prometheus + Grafana
- **Infrastructure**: Docker Compose → Kubernetes (K3s)
