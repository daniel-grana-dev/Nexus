# Nexus

Self-hosted AI inference platform with hybrid LLM routing, RAG pipeline, and production monitoring.

> Estado actual: Configuración inicial — Día 1 del plan de 12 meses.

## Stack
- **LLM local**: Ollama (Llama 3.2)
- **LLM cloud**: OpenAI GPT-4o-mini
- **API**: FastAPI + Python
- **Base de datos**: PostgreSQL + pgvector
- **Monitorización**: Prometheus + Grafana
- **Infraestructura**: Docker Compose → Kubernetes (K3s)
